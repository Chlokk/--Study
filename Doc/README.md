### 后端

基于maven进行项目构建，并且进行分模块开发

**sky-take-out**是父工程：统一管理依赖版本，聚合其他子模块

其下包括三个子工程，分别为：

#### 1. sky-common

存放公共类，例如：工具类，常量类，异常类等

#### 2. sky-pojo

*(Plain Old Java Object,普通旧java对象)*

存放实体类,VO,DTO等

#### 3. sky-serve

后端服务，存放配置文件，Controller，Service，Mapper等
